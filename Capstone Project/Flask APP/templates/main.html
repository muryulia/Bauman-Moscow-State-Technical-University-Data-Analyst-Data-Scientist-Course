Depth and Width of the Welded Joint:
Forecasting Model

The model predicts parameters of depth and width of a welded joint by applying the machine learning method of Gradient Boosting Regressor. The Gradient Boosting predicts two variables at a time by using MultiOutputRegressor.

We used the R-squared metric to evaluate forecasting accuracy. The metric returns 0.94 for the depth of the joint and 0.95 for the width of the joint, and its mean cross-validation score for two targets is 0.96 on the train set and 0.92 on the test set.

The model is a part of a capstone project for the Data Analyst (Data Scientist) Course by Bauman Moscow State Technical University.
